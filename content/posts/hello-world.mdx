# Hello World

Welcome to the **Software Development Blog**. This is a sample post to demonstrate all the features of the blog.

## Features

The blog supports:

- **Headings** at various levels
- **Code blocks** with syntax highlighting
- **Inline code** like `console.log()`
- **Links** to [external resources](https://example.com)
- **Lists** (both ordered and unordered)
